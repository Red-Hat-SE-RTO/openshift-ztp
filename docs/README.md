# Quick Start Guide

### Requirements 
* ODF 
* OPENSHIFT

### Run bootstrap-hub.sh
**Log into the Hub cluster with a cluster-admin user:**
```
oc login ...
```
**Start bootstrap-hub.sh**
```
./bootstrap-hub.sh
```
